# 🦔 穿山甲的回家之路 — 保育互動遊戲

臺灣二級保育類動物穿山甲的互動式保育教育遊戲，適合國小高年級學童。

## 遊戲特色

- **10 道情境關卡**：認識穿山甲、夜間覓食、遭遇流浪狗、穿越馬路、捕獸鋏陷阱、媽媽的尾巴、發現受傷穿山甲、政府源頭管理政策等
- **即時回饋**：答對/答錯附詳解 + 小知識
- **音效與動畫**：Web Audio API 音效、彩帶慶祝動畫
- **倒數計時**：每題 15 秒，增加挑戰性
- **無需安裝**：純 HTML，瀏覽器直接開啟即可玩

## 如何遊玩

直接開啟 `pangolin-game.html` 即可。

## 教材對應

- 科目：自然科學
- 主題：臺灣二級保育類動物 — 穿山甲
- 適用年級：國小 5-6 年級
- 學習目標：認識穿山甲生態習性、了解保育威脅、學習正確保育行動
